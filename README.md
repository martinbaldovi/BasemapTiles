# BaseMap Plugin for QGIS 4.0

Add popular basemap tiles (OpenStreetMap, Esri Satellite, Google Maps, Google Satellite, Bing Maps, Bing Satellite) to the layer list with one click.

## How to install

1. Download the source code as a ZIP archive.
2. Uncompress the ZIP file.
3. Move the extracted `BasemapTiles` parent folder to the QGIS 4 plugins directory (create folders if they don't exist):  
   `%APPDATA%\QGIS\QGIS4\profiles\default\python\plugins`

## How to use

1. Load a vector layer in QGIS 4.0.
2. Enable **BasemapTiles** from the **Plugin Manager**.
3. Click **➕ BasemapTiles** and choose basemap tiles.
4. The selected basemap tiles will be displayed in the workspace.
